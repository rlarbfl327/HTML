HTML5 입력 양식 태그와 구조화 태그


01 다음 괄호 안을 채우시오.

  • ( )은 사용자에게 정보를 입력받는 요소입니다.
  • form 태그는 ( )속성의 방식으로 ( )속성 장소에 데이터를 전달합니다.
  • ( GET )방식은 주소에 데이터를 입력해서 전달합니다.
  • ( POST )방식은 GET 방식과 달리 비밀스럽게 데이터를 전달합니다.
  • input 태그에 ( )속성을 지정해서 다양한 종류의 기본 입력 양식을 생성합니다.
  • 라디오 버튼의 ( )속성을 사용하면 여러 대상 중 하나만 선택하는 형태를 구현할 수 있습니다.
  • ( )태그는 목록으로 보여 주는 항목 중 하나 또는 여러 개를 선택할 때 사용하는 입력 양식 요소입니다.
  • div 태그는 공간을 ( )형식으로 분할합니다.
  • span 태그는 공간을 ( )형식으로 분할합니다.
  • ( )은 컴퓨터 프로그램이 코드를 읽고 의미를 인식할 수 있는 지능형 웹을 의미합니다.
