  3. CSS도 HTML처럼 <!-- 주석 --> 형태로 주석을 사용합니다. ( )
  4. 2em은 2배를 나타내는 크기 단위입니다. ( )
  5. 색상 단위에서 투명도를 지정할 때는a rgb( ) 등을 사용합니다.( )
